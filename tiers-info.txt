Тиры:
Bronze | Silver | Gold | Platinum

Скидки:
4%     | 6%     | 8%   | 10%

Минимальный взнос для получения тира:
1000   | 3000   | 5000 | 10000

Описание:
- Первая строка — названия тиров.
- Вторая строка — скидки для каждого тира.
- Третья строка — минимальный объём застейканных токенов для получения соответствующего тира.

Формула расчёта тира:
Тир пользователя определяется по интегральному коэффициенту значимости (K):

K = 0.65 * (T / T_max) + 0.35 * (A / A_max)

Где:
- T — время стейкинга пользователя (в днях)
- T_max — максимальное время стейкинга среди всех пользователей (в днях)
- A — количество застейканных токенов пользователя
- A_max — максимальное количество застейканных токенов среди всех пользователей

Пороговые значения K для получения тира:
- Bronze:   0 ≤ K < 0.25
- Silver:  0.25 ≤ K < 0.5
- Gold:    0.5 ≤ K < 0.75
- Platinum: 0.75 ≤ K ≤ 1

Таким образом, время стейкинга влияет на итоговый тир сильнее, чем количество токенов, что стимулирует долгосрочное участие.
